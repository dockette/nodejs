## Usage

### Node v9

- Node: `9.1.0`
- NPM: `5.4.2`

```
docker run -v /path/to/site:/srv dockette/nodejs:v9
```

### Node v8

- Node: `8.9.1`
- NPM: `5.0.3`

```
docker run -v /path/to/site:/srv dockette/nodejs:v8
```

### Node v7 

- Node: `7.10.1`
- NPM: `4.2.0`

```
docker run -v /path/to/site:/srv dockette/nodejs:v7
```
